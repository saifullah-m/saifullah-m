## 🚀 Deploy — GitHub Pages Setup

The portfolio site lives in the [`docs/`](docs/) folder of this repository and is deployed via **GitHub Pages**.

### Enable GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under **Source**, choose:
   - **Branch:** `main`
   - **Folder:** `/docs`
3. Click **Save**.
4. After a minute or two your site will be live at:
   ```
   https://saifullah-m.github.io/Saifullah-1/
   ```

### Preview locally

No build step required — it's plain HTML/CSS/JS.

```bash
# Clone the repo
git clone https://github.com/saifullah-m/Saifullah-1.git
cd Saifullah-1

# Serve the docs/ folder with any static server, e.g.:
npx serve docs
# or
python3 -m http.server 8080 --directory docs
```

Then open <http://localhost:8080> (or whichever port the server reports).

### Edit the site

| File | Purpose |
|---|---|
| `docs/index.html` | Page content — sections, text, links |
| `docs/css/style.css` | All styles + theme variables |
| `docs/js/main.js` | Dark/light toggle, mobile nav |

To add a real project, open `docs/index.html` and find the **Projects** section — comments in the HTML guide you through duplicating a card and filling in your details.
